var onReleaseRequestState(default,null) : hxevents.AsyncDispatcher<HttpApplication>
Occurs after ASP.NET finishes executing all request event handlers. This event causes state modules to save the current state data.
var onResolveRequestCache(default,null) : hxevents.AsyncDispatcher<HttpApplication>
Occurs to let the caching modules serve requests from the cache, bypassing execution of the event handler (for example, a page or an XML Web service).
var onUpdateRequestCache(default,null) : hxevents.AsyncDispatcher<HttpApplication>
Occurs when an event handler finishes execution in order to let caching modules store responses that will be used to serve subsequent requests from the cache.
